EPIC-001 - Feature Prompt File

This epic builds the basic game structure for the football career game. The goal of this first implementation pass is to create the core screen flow and the first part of the career setup process.

Scope for this prompt file:
- Replace the current landing flow with a new Home screen.
- Add the full route structure for the main game pages.
- Build the Career Start screen shell.
- Implement editable Team Name and Team Stadium fields.
- Prepare state for Team Kit and Players, but do not build those creation flows yet.
- The button to start the career must remain disabled until all required team values exist, including Team Kit and Players.

Important scope rules:
- Do not implement Team Kit creation yet.
- Do not implement Player selection/generation yet.
- Do not allow progression into the career until Team Name, Team Stadium, Team Kit, and Players are all present in state.
- Team Kit and Players can exist as placeholder/null state for now, but they must be part of the validation logic.
- The layout should span the whole page width and be responsive.
- This should feel like the real game flow, not a debug flow.
