What was implemented:
- Refactored `LeagueTablePanel` into explicit variants:
  - `detailed`: preserves full standings columns, form dots, team click -> squad modal
  - `compact`: shows only `Pos`, `Team`, `Pts` and supports full-card link mode
- Added controlled competition support to `LeagueTablePanel` so League Stats tabs can drive which league table is displayed.
- Added compact panel link treatment and compact-table sizing for controls-panel use.
- Updated Career Home flow:
  - removed standalone full-width league-table panel
  - inserted compact snapshot inside `CareerControlPanel` below navigation
  - wired snapshot to player league by default and linked entire card to `/career/league-stats`
- Updated League Stats page:
  - added one tab per domestic league
  - only selected league is rendered at a time
  - top section shows detailed `LeagueTablePanel`
  - player stat tables shown directly below in forced 3-column layout
- Updated shared player-stat table component to support optional fixed 3-column grid mode.
- Added developer-focused debug block in League Stats (dev mode) for selected league + row-count visibility.
