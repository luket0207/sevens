What was implemented:
- Wizard structure update:
  - Replaced tab flow with:
    - Name and Stadium Name
    - Kits
    - Team Selection
    - Coaches
  - Removed standalone Players tab from navigation and rendering.
- Coach selector system:
  - Added coach selector state shape with `choiceGroups`, `selectedByGroup`, `isComplete`.
  - Added generation utility that creates 2 stable groups of 3 common staff member coach cards.
  - Added validation helpers to:
    - read selected coaches from selector state
    - determine selector completion
    - validate career starting coaches for setup completion.
  - Added Coaches tab UI that:
    - generates/regenerates coach options
    - enforces one selected coach per group
    - shows progress (0/2, 1/2, 2/2)
    - stores selected coaches in `career.setup.coaches` and selector state in `career.setup.coachSelector`.
- Coach attribute presentation:
  - Reused `PlayerSkillsetBars` and extended it for coach stats.
  - Added coach stat ordering keys:
    - Overall Rating, Scouting, Judgement, Youth Training, GK/DF/MD/AT Training
  - Added `hideTraits` prop so coach cards display stat bars without trait section.
- Team Selection merge and gating:
  - Team Selection tab now contains two phases:
    - Phase 1: existing player generation and selection flow (`CareerTeamSelector`)
    - Phase 2: pitch placement and tactic setup
  - Phase 2 remains hidden until player selection is complete.
  - Added random auto-place action in Phase 2:
    - shuffles selected outfield players into valid slots
    - goalkeeper stays fixed via setup state.
- Setup and validation integration:
  - Career setup validation now requires valid starting coaches.
  - Team Selection tab completion now reflects merged validity:
    - player selection complete
    - team placement complete.
  - Added lightweight debug visibility for:
    - selected coach count/names
    - coach selection completion
    - player selection completion
    - team placement completion
    - merged Team Selection completion
    - full wizard validity.
- Generation rebalance + persistence:
  - Tightened outfield weighted overall pool to 20/21/22 only.
  - Persisted selected starting coaches to generated player-team state:
    - `playerTeam.coaches`
    - `playerTeam.staff.coaches`.
