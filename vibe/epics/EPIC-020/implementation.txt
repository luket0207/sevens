Feature pass (2026-03-13, FEAT-2001 to FEAT-2013):
- Implemented the full Academy card usage flow from the updated `EPIC-020/features.txt`.
- Key updates:
  - Added `Go to Academy` actions for Academy cards on Career Home without consuming the card.
  - Added a dedicated Academy card bar above academy players with horizontal scrolling and reusable card tiles.
  - Added centralized Academy card effect resolution for:
    - all-player maturity reduction
    - single-target maturity reduction
    - academy slot expansion with max-capacity fallback
    - current value reveal
    - potential value reveal
  - Added explicit best-current-staff `Judgement` sourcing for Academy reveal cards.
  - Added persistent academy value-reveal state for `Current Value` and `Potential Value`.
  - Added single-target choose mode on the Academy page and ensured cards are consumed only after successful application.
  - Added persistent academy slot expansion state and updated scouting/academy/home capacity checks to use expanded capacity.
  - Preserved academy reveal metadata when players are promoted into the first team.
  - Added academy-card debug payloads to academy/card state for effect rolls, reveal outcomes, judgement source, slot changes, and maturity reductions.
