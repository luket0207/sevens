EPIC-008 - Team Management Page

Overview
The goal of this epic is to build the Team Management page.

This page allows the player to:
- arrange their 6 outfield players onto the pitch
- keep the goalkeeper fixed in goal
- set defensive and attacking tactics
- calculate the team's tactic ratings:
  - DTR = Team Defence Tactic Rating
  - ATR = Team Attack Tactic Rating
- save or cancel their changes and return to the calendar

This epic is about:
- squad arrangement UI
- drag/drop interaction
- tactic selection
- tactic rating calculation
- save/cancel flow

This epic is not about:
- match simulation
- width logic during matches
- opponent tactics
- stamina, injuries, suspensions, or substitutions
- validating match outcomes from tactics

================================================================
Core Page Layout
================================================================

The Team Management page should have 3 main areas:

1. Tactic Box at the top
- contains defensive tactic controls
- contains attacking tactic controls

2. Main team management area below
- player list on the left
- pitch on the right
- pitch takes the main visual focus
- player list allows drag/pick-up behaviour

3. Bottom or final action area
- Save button
- Cancel button

General layout behaviour:
- the player list should be to the left of the pitch
- the pitch should be on the right
- the goalkeeper is fixed in goal
- there are 6 outfield pitch slots
- players can be moved from the list to the pitch
- players can be moved from the pitch back to the list or dropped off the pitch to remove them from the pitch
- the page should be responsive, but can prioritise desktop layout first

================================================================
Team Arrangement Rules
================================================================

The team consists of:
- 1 goalkeeper
- 6 outfield players

Goalkeeper rules:
- the goalkeeper is fixed in goal
- the user does not drag the goalkeeper around
- the goalkeeper should always be visible in the goal slot

Outfield player rules:
- there are 6 pitch slots for outfield players
- the user can place any outfield player into any of the 6 slots
- players do not need to be assigned a strict football position here
- the user is simply arranging their chosen team on the pitch

Interaction rules:
- players can be picked up from the list and placed on the pitch
- players can be picked up from the pitch and moved to another pitch slot
- players can be picked up from the pitch and dropped off the pitch to remove them from the pitch
- if a player is removed from the pitch, they return to the available pool/list state
- a player should not exist in two places at once

Team completeness rule:
- when all 6 outfield slots are filled, the team is complete
- only a complete team can be saved
- until then, the player can only exit without saving

================================================================
Page Navigation Rules
================================================================

This page only needs 2 navigation buttons:
- Save
- Cancel

Save:

- if confirmed:
  - save the team changes

  - return to the calendar page

Important behaviour:

- Save persists changes
- Cancel does not persist changes

================================================================
Tactic Rating Outputs
================================================================

This epic must calculate 2 values:

- Team Defence Tactic Rating (DTR)
- Team Attack Tactic Rating (ATR)

Rules:
- both are whole numbers
- both should be rounded to the nearest whole number
- both are final values between 1 and 100
- for now they should be output for debugging/inspection

Calculation process:
1. work out the relevant skill sums by player group
2. apply the tactic matrix formulas for the selected tactic
3. add all formula results in that tactic column
4. divide the total by 10
5. round to a whole number

Reasoning:
- each tactic column has an intended maximum of 1000 before dividing by 10
- some columns intentionally will not be able to reach 1000 even with perfect skills
- after dividing by 10 and rounding, the result becomes the 1 to 100 rating scale

================================================================
Player Group Values Used in Calculations
================================================================

Use these grouped totals:

DF = Sum of the team's defenders' ratings for a given skill
MD = Sum of the team's midfielders' ratings for a given skill
AT = Sum of the team's attackers' ratings for a given skill
ALL = Sum of the whole team's ratings for a given skill

Important note:
- these groups refer to the players placed into the team shape on this page
- the system must know which pitch slots count as defenders, midfielders, and attackers
- even though the user is just dragging players onto 6 slots, the 6 pitch slots themselves must carry the role meaning needed for calculations
- therefore the pitch layout should define:
  - 2 defender slots
  - 2 midfielder slots
  - 2 attacker slots
- the goalkeeper remains fixed and is not part of DF, MD, or AT unless a formula explicitly uses ALL and the agreed implementation includes goalkeeper skill values
- for this epic, use only the 6 outfield players for DF/MD/AT/ALL in the tactic calculations unless an existing design elsewhere explicitly requires goalkeeper inclusion

Recommended slot structure:
- 2 defender slots
- 2 midfielder slots
- 2 attacker slots

This keeps the drag/drop free enough for the user while still making tactic calculations meaningful.

================================================================
Tactic Contribution Matrix
================================================================

Use the following exact matrix.

A "/" means no contribution for that skill/tactic combination.

Defence (DTR) and Attack (ATR) matrix:

	Defence (DTR)					Attack (ATR)
	Low Block	Mid Block	High Press	Offside Trap	Zonal	Posession	Long Ball	Wing Play	Counter	Direct
Tackling	DF * 1.5	DF + MD	ALL / 3	/	DF * 2	/	/	/	/	/
Marking	DF * 1.5	DF + MD	ALL / 3	DF * 2	ALL / 3	/	/	/	/	/
Passing	/	/	/	/	/	MD + ((DF + AT) / 2)	(MD * 2) + DF	MD	(DF + (MD * 2)) / 2	MD
Movement	/	/	ALL	DF / 2	/	MD + ((DF + AT) / 2)	AT	AT	AT + MD	/
Control	DF	(DF + MD) / 2	/	DF * 2	ALL / 2	MD	/	/	/	/
Dribbling	/	/	/	/	/	/	/	MD	(MD + (AT * 2)) / 2	AT + MD
Shooting	/	/	/	/	/	/	AT	AT * 2	/	AT + MD

Important correction:
- where earlier wording used "MF", implement this as "MD"
- the midfielder group variable used throughout should be MD

Example:
If Tackling is being evaluated for Low Block:
- formula = DF * 1.5

If the two defender slot players have Tackling 20 and 30:
- DF = 50
- contribution = 50 * 1.5 = 75

================================================================
DTR and ATR Calculation Rules
================================================================

Defensive side:
- take the selected defensive tactic
- read the column for that tactic from the Defence matrix
- evaluate every non-empty skill formula in that column
- add those results together
- divide by 10
- round the result
- output as DTR

Attacking side:
- take the selected attacking tactic
- read the column for that tactic from the Attack matrix
- evaluate every non-empty skill formula in that column
- add those results together
- divide by 10
- round the result
- output as ATR

For now:
- output DTR and ATR clearly for debugging
- no further match logic is needed yet
