EPIC OVERVIEW
Last updated: 2026-03-13

EPIC-001 - Core Career Shell (Status: done)
This epic established the first playable career shell by replacing the original landing flow with a new Home page and wiring base career routes. It implemented the Career Start page structure with required setup state (team name, stadium, kit, players) and gating so career start remains disabled until required values are present.

EPIC-002 - Team Kit System and Temporary Generator Layer (Status: done)
This epic delivered the initial team kit pipeline, including reusable shirt pattern modeling, kit selectors, goalkeeper kit support, and visual shirt rendering from saved data. It also introduced a temporary player-generation debug path to exercise generation mechanics, then removed that temporary path after bug-fix iterations to prepare for the full generator rebuild.

EPIC-003 - Player Generation Rebuild (Status: done)
This epic rebuilt player generation into a reusable module with explicit goalkeeper/outfield models, constrained skill outputs, and exact average-based overall behavior. It added influence-rule logic and deterministic generation utilities designed for reuse in later systems, with debug controls used for verification.

EPIC-004 - Career Team Selector (Status: done)
This epic implemented the staged team-selection flow on Career Start (1 goalkeeper + 6 outfield picks from grouped choices). It added pool generation, forced influence spread rules, progression/validation behavior, and selected-player display cards. Follow-up bug fixes refined display, interaction, and rounding consistency.

EPIC-005 - Career Generation Pipeline and World Build (Status: done)
This epic introduced full career world generation after setup, including generation loading/progress phases and persistence to career state. It generates the football world structure (competitions, teams, players, kits) and transitions into Career Home with debugging support. Bug fixes focused on generation reliability and flow stability.

EPIC-006 - Reusable Player Image Component (Status: done)
This epic added a reusable player image renderer driven by generated appearance data and kit inputs. It covered appearance parsing, layered rendering for visual traits, and integration points for existing player displays, plus debug/gallery support.

EPIC-007 - Career Calendar and Main Loop Foundation (Status: done)
This epic converted Career Home into a calendar-driven season loop with day progression, month navigation, event scheduling, and foundational cup/league calendar structure. It added core control flow and debug visibility for day advancement and schedule state.

EPIC-008 - Team Management Page (Status: done)
This epic replaced the Team Management placeholder with a complete squad-arrangement and tactics page, including drag/drop outfield placement, fixed goalkeeper, tactic selection, and DTR/ATR/TC calculations. It persisted team-management state back into career data and received extensive UX/logic bug fixes (layout, list behavior, skill bars, compatibility table updates).

EPIC-009 - Manager Generation and Team Assignment (Status: done)
This epic added procedural manager generation for AI teams and persisted manager tactical preference data in career world state. It integrated manager assignment into generation flow, added progress/debug visibility, and later fixed manager identity by generating manager names.

EPIC-010 - League/Cup Simulation and Career Progression Flow (Status: done)
This epic added league-table simulation, match result generation, cup progression, season outcomes, and player/AI day-flow handling across Career Home, Match, Day Results, and Cup Draw pages. It also introduced league/cup stats views, season-summary transition flow, and many simulation/UI bug fixes. Recent fixes include updated MRD result matrices, corrected player-team match-profile rating input source, and stricter cup-draw continue gating until reveals are complete.

EPIC-011 - Calendar and Career Flow Tidy Up (Status: done)
This epic focused on UX polish for calendar and progression flow: context-aware Continue labels, day-one setup gate behavior, icon-and-colour calendar events, improved tooltip/event display rules, and fixture/draw formatting refinements. Bug fixes addressed playoff visibility filtering and tooltip-trigger styling/sizing consistency.

EPIC-012 - Career Start Wizard and Shared Team Setup Layout (Status: done)
This epic converted `/career/start` into a 4-tab wizard (Name/Stadium, Kits, Players, Team Selection) with free tab navigation and globally accessible Start Career actions. It moved the initial required team arrangement/tactics setup into pre-career flow and introduced a shared top-strip + side-on pitch layout reused by both Career Start Team Selection and Team Management. The duplicate in-career day-1 team setup gate was removed, with pre-career setup now persisted into generated player-team state.

EPIC-013 - Player Traits (Status: done)
This epic introduced a full player-traits foundation across generation, storage, and UI display. It added weighted trait assignment (0/1/2 trait distribution), influence-context rarity mapping (including goalkeeper-specific weighting), and trait persistence for both user and AI players. Traits are now shown in shared player skillset views while remaining data-only and explicitly excluded from overall/tactic calculations.

EPIC-014 - Roguelike Card Foundation (Status: done)
This epic replaced the prior training-first direction with a complete card reward/library backbone for career mode. It delivered the unified card model, full definition tables, reward matrix + resolver, 3-card mandatory choice flow, reroll-by-discard logic, persistent card library (sort/filter/discard), and a dedicated reward route. Follow-up bug fixes added debug reward-input controls and stabilized card tile layout with fixed-height, aligned card rows.

EPIC-015 - Career Start Coaches Tab, Team Selection UX Tidy Up, and Team Selection Rebalance (Status: done)
This epic reworked `/career/start` into a cleaner flow by removing the standalone Players tab, merging player choice into Team Selection, adding coach selection (2 picks from 2x3 common coach groups), and gating pitch setup until player selection is complete with random auto-place support. It also tightened initial team generation from 20-25 to 20-22, integrated coach data into career state and staff page display, and included iterative UX/polish fixes across wizard navigation, kit setup, and setup completion flow.

EPIC-016 - Button System Visual Refresh (Status: done)
This epic refreshed the shared button system across primary/secondary/tertiary variants with centralized design tokens, stronger visual hierarchy, and premium hover/focus/active/disabled states without layout movement. It also added a dedicated contextual Continue action control for the career calendar with event-based icon/colour treatment and bespoke styling tuned through iterative bug-fix passes.

EPIC-017 - League Table Layout Split and League Stats Tabs (Status: done)
This epic split league-table rendering into compact and detailed variants, moved the compact snapshot into the Career Home controls panel, and made it a full-card link to League Stats. It also reworked League Stats into one-league-at-a-time tabs with the detailed table at the top and 3-column player stat tables below, while preserving team-click squad modal behavior in detailed mode.

EPIC-018 - Staff Card Usage, Staff Slots, and Staff Progression (Status: done)
This epic made staff cards fully usable by adding staff-member hiring, full-capacity replacement/sacking flow, staff-upgrade application, slot unlocking from first-time promotions, and 7-day expiry for unused staff-member cards. It also integrated staff-card state with calendar progression, cleaned up staff data/visuals through follow-up bug fixes, and added lightweight debug support for validating slot, expiry, and hire/use behavior.

EPIC-019 - Scouting Card Usage, Scouting Reports, and Academy Foundation (Status: done)
This epic made scouting cards fully playable by adding staff assignment, in-use staff locking, calendar scouting markers, scouting report generation, revealable scouting intel, academy storage, maturity progression, promotion flows, and mature-player loss rules. Follow-up feature and bug passes added coach recruitment recommendations, academy replacement handling, responsive report/card polish, and recommendation tuning including the new low-confidence `Hunch` layer.
